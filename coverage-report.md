# Reporte de Cobertura de Pruebas Unitarias

Este documento contiene el reporte de cobertura de las pruebas unitarias generadas por JaCoCo.

## Instrucciones para Generar el Reporte
1. Ejecutar `mvn clean test` para ejecutar las pruebas y generar el reporte.
2. El reporte estará disponible en `target/site/jacoco/index.html`.

## Objetivo
Asegurar que al menos el 80% del código esté cubierto por las pruebas unitarias.

## Herramientas Utilizadas
- JUnit 5 para las pruebas unitarias.
- JaCoCo para la verificación de cobertura.
